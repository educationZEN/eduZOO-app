Der Begriff der Menge allein reicht nicht aus, um die Welt zu beschreiben. Wir können jetzt zwar eindeutig sagen, welche Personen in Berlin leben, um aber jedem Berliner einen Wohnort zu geben, müssten wir uns mit vielen Teilmengen auseinandersetzen. Der Vorteil, den uns der Mengenbegriff verschafft hat wäre dahin. Es gibt jedoch die Möglichkeit, den Elementen der Menge der Berliner (also den Berlinern) Koordinaten zuzuordnen. Dies leistet eine Abbildung.

Seien M,N Mengen. Eine Abbildung oder Funktion von M nach N ist ein Tripel (M,N,f), bestehend aus dem Definitionsbereich M, dem Wertebereich N und der Zuordnungsvorschrift f, die jedem Element von M genau ein Element aus N zuweist. Für x\in M bezeichnet f(x) das Element aus N, das x zugeordnet wird. Die Abbildung wird notiert als f:M\to N, x\mapsto f(x) Das Element f(x) heißt Bild von x unter f, das Element x heißt Urbild von f(x).

Die Zeile f:M\to N, x\mapsto f(x) wird gesprochen "f von M nach N, x wird abgebildet auf f von x". Wesentlich an der Definition ist, dass jedes Urbild genau ein Bild besitzt und nicht mehrere. Eine Person hält sich an genau einem Ort auf.
Sind Definitions- und Wertebereich klar, so wird statt (M,N,f) schlicht f geschrieben.

Es bezeichne \mathcal{L} die Menge aller Längengrade, \mathcal{B} die Menge aller Breitengrade, \mathcal{O}=\mathcal{L}\times\mathcal{B} die Menge aller Koordinaten der Orte auf der Erde, \mathcal{P} die Menge aller Personen, \omega die Abbildung, die jeder Person die Koordinaten ihres momentanen Aufenthaltsorts zuweist. Dann ist \omega:\mathcal{P}\to\mathcal{O}\; ,\; p\mapsto\omega(p)=(l,b) eine Abbildung, da sich an bestimmten Koordinaten immer nur eine Person befinden kann. Jedoch definiert \phi:\mathcal{O}\to\mathcal{P}\; ,\; (l,b)\mapsto\phi(l,b)=p keine Abbildung, da nicht jedem Ort notwendigerweise eine Person zugeordnet werden kann.

Eine Abbildung ordnet jedem Element des Definitionsbereichs höchstens ein Element des Wertebreichs zu. Alle Elemente des Definitionsbereichs müssen abgebildet werden, aber nicht alle Elemente des Wertebereichs müssen ein Urbild haben.

Injektivität, Surjektivität, Bijektivität

Häufige Problemstellung beinhalten das Lösen von Gleichungen f(x)=y für eine gegebene Funktion f. Gibt es ein x im Definitionsbereich, das die Gleichung erfüllt, so heißt x Lösung der Gleichung. Ist dieses x eindeutig, so lässt sich die Gleichung auflösen nach x, mit dem funktionalen Zusammenhang x=g(y), wobei g eine neue Funktion ist, die sogenannte Inverse von f. Im Folgenden werden die Begriffe eingeführt, die notwendig sind, um die inverse Funktion zu zu definieren.

Eine Abbildung f:\mathcal{D}\to\mathcal{W} heißt injektiv, genau dann wenn jedes Element des Wertebereichs höchstens ein Urbild besitzt.

Das bedeutet nicht, dass jedes Element des Wertebereichs ein Urbild besitzt. Es bedeutet, dass wenn ein Element des Wertebereichs ein Urbild besitzt, so ist dieses eindeutig. Streng formuliert ist eine Abbildung f:\mathcal{D}\to\mathcal{W} also injektiv, falls aus f(x)=f(y) folgt, dass x=y.

Eine Abbildung f:\mathcal{D}\to\mathcal{W} heißt surjektiv, genau dann wenn jedes Element des Wertebereichs ein Urbild besitzt.

Hier ist der Unterschied im Wortlaut der Definitionen von Injektivität und Surjektivität zu beachten. Während bei Injektivität jedes Element des Wertebreichs höchstens ein Urbild hat, so hat bei Surjektivität jedes Element des Wertebereichs mindestens ein Urbild. Ist eine Funktion also injektiv und surjektiv zugleich, folgt, dass jedes Element des Wertebereichs genau ein Urbild hat, die Zuordnung also nicht nur eindeutig, sondern eineindeutig ist.

Eine Abbildung f:\mathcal{D}\to\mathcal{W} heißt bijektiv, genau dann wenn sie injektiv und surjektiv ist. In diesem Fall definiert f^{-1}:\mathcal{W}\to\mathcal{D}\;,\;y=f(x)\mapsto f^{-1}(y)=x eine Abbildung, die Umkehrabbildung oder Inverse von f

Der Wertebereich einer Funktion kann immer so gewählt werden, dass die Funktion surjektiv ist. Die Menge f(\mathcal{D})=\{f(x)|x\in\mathcal{D}\} ist die Menge aller Bilder und beschert als Wertebereich einer Funktion immer Surjektivität.

Die Inverse f^{-1} wird nicht "f hoch minus eins" gesprochen sondern "f invers". Es ist zu beachten, dass f^{-1} im Allgmeinen nicht den Kehrwert der Funktion f darstellt, also f^{-1}(x)\neq\frac{1}{f(x)} im Allgemeinen.
